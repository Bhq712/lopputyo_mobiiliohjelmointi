# Mobiiliohjelmointi lopputyö
## Elokuvien ja sarjojen arvostelusovellus
## Käytetyt teknologiat:
* Käyttäjä pystyy arvostelemaan eri elokuvia ja sarjoja sekä lisäämään niitä suosikeikseen.
* Käyttäjä pystyy käyttämään laitteen kameraa ja galleriaa profiilikuvaa varten.
* Käyttäjä pystyy etsimään ja selaamaan API:n kautta eri elokuvia ja sarjoja kirjastosta, ja näkee niistä perustietoja.
* Sovelluksessa on käytössä drawer ja stack -navigointi.
* Käyttäjä pystyy kirjautumaan sisään sovellukseen ja ulos.
